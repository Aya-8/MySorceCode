# 昔の労働ゲーム(煮熟ゲーム)

インターンシップ応募用に、昔の労働ゲーム(煮熟ゲーム)で自分が書いたスクリプトをまとめたフォルダです。

## 使用技術

- Unity
- C#

## 収録内容

- `Controller`: 入力、プレイヤー操作、時間、サウンド、UI 表示関連
- `Director`: ゲーム進行、フェーズ管理関連
- `Enums`: 開始シーケンスなどの列挙型関連
- `Pole`: 棒の移動アニメーション関連
- `InputChecker.cs`: 入力チェック関連

## 元プロジェクト内の場所

- `Assets/Kapuro2025Winter/Scripts/Stiring`
